Instrucciones

🇪🇸 Guía de Sintaxis Temporal

Añade estos comandos en cualquier parte de tu párrafo para alterar el flujo:

  • (P:2s) Pausa: Congela el texto durante 2 segundos. Ideal para interludios.
  • (V:1.5x) Velocidad: Multiplica la velocidad del párrafo (1.5x es más rápido, 0.5x es lento).

🎙️ Modo Estándar

Texto fluido. El motor calcula el peso de cada palabra automáticamente.

🗣️ Mapeo Fonético (Shadowing)

Usa la barra vertical | para separar la escritura formal de la reducción fonética. NUEVO: ¡Puedes adjuntar una Pista Base (mp3, wav) desde el Gestor para escuchar al hablante nativo mientras ensayas!

Ej: ¿Para dónde vas? | ¿Pa'nde vas?

🎵 Modo Musical

Notación americana (*ChordPro*). Escribe la nota entre corchetes [] antes de la sílaba.

Ej: [C4]La don[C4]na è [F4]mobile (P:2s)

Nota de Grabación (Metrónomo / Pista Base): Si reproduces audio de fondo mientras grabas sin auriculares, tu voz podría sonar robótica debido al filtro de cancelación de eco del teléfono. Usa auriculares o silencia la pista base con el botón 🔊 superior para obtener grabaciones profesionales.

🇺🇸 Temporal Syntax Guide

Add these tags anywhere in your paragraph to alter the flow:

  • (P:2s) Pause: Freezes scrolling for 2 seconds. Great for interludes.
  • (V:1.5x) Velocity: Multiplies paragraph speed (1.5x is faster, 0.5x is slower).

🎙️ Standard Mode

Fluid reading. The engine automatically calculates word density.

🗣️ Phonetic Mapping (Shadowing)

Use the vertical bar | to separate formal text from its slang/phonetic reduction. NEW: You can attach a Base Track (mp3, wav) in the Manager to listen to the native speaker while rehearsing!

Ex: Where are you going? | Where ya goin'?

🎵 Musical Mode

Scientific ChordPro notation. Write the chord inside square brackets [] before the syllable.

Ex: [C4]Twin[C4]kle, [F4]little [C4]star (P:2s)

Recording Note (Metronome / Base Track): If playing background audio while recording without headphones, your voice might sound robotic due to the phone's built-in echo cancellation. Use headphones or mute the base track with the top 🔊 button for professional recordings.
