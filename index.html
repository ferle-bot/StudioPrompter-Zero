🇪🇸 Guía de Uso Rápido

Bienvenido a StudioPrompter Zero, un teleprompter 100% local y privado. Soporta 3 modos de escritura:

1. Modo Estándar (🎙️)

Escribe tu texto normalmente. El sistema calculará la velocidad de lectura basándose en la longitud de las palabras y los signos de puntuación.

2. Modo Shadowing (🗣️)

Ideal para practicar idiomas, doblaje o fonética. Usa la barra vertical | para separar el texto formal de la pronunciación.

Ejemplo: Hello world | Jelou uorld

3. Modo Musical (🎵)

Formato tipo ChordPro visual. Escribe el acorde o nota entre corchetes rectos [] justo antes de la sílaba donde debe cantarse/tocarse.

Ejemplo: [C4]La don[C4]na è [F4]mobile

🇺🇸 Quick Start Guide

Welcome to StudioPrompter Zero, a 100% local and private teleprompter. It supports 3 writing modes:

1. Standard Mode (🎙️)

Write your script normally. The system will calculate the reading speed based on word length and punctuation.

2. Shadowing Mode (🗣️)

Perfect for language learning or voice acting. Use the vertical bar | to separate formal text from pronunciation.

Example: Hello world | Jelou uorld

3. Musical Mode (🎵)

Visual ChordPro format. Write the chord or note inside square brackets [] right before the syllable where it should be played.

Example: [C4]La don[C4]na è [F4]mobile
