# StudioPrompter-Zero 🎙️📱

## English

**StudioPrompter-Zero** is a high-performance, **Local-First** Progressive Web App (PWA) designed for content creators, podcasters, and video producers. It provides a professional-grade studio recording and teleprompter environment optimized for mobile devices, operating with zero external server dependencies, zero cost, and total data privacy.

### 🚀 Key Features
* **Fixed-Focus Ergonomic Layout:** A static, three-row high-contrast reading interface. It eliminates vertical eye tracking, reducing visual fatigue and ensuring a natural look directly at the camera.
* **Precision Autoscroll Engine:** Micro-calibrated speed adjustments in strict **0.01** increments driven by `requestAnimationFrame` for stutter-free 60 FPS scrolling.
* **Local Audio Studio (Web Audio API):** Built-in adaptive noise gate and hardware-accelerated pass-band filtering. Includes an automatic **crossfade stitch feature** that seamlessly blends audio fragments when pausing and resuming recording.
* **Chroma Key Sync (Modo Cine):** Sintonizes recorded audio playback with horizontal text scrolling over a solid green background (`#00FF00`). Perfect for capturing your screen and overlaying dynamic subtitles in video editors like YouCut.
* **100% Private & Serverless:** All data, text scripts, and audio takes remain strictly inside the device's secure sandbox (IndexedDB / OPFS) with an option to export files directly to an external SD card.

## Español

**StudioPrompter-Zero** es una Progressive Web App (PWA) de alto rendimiento diseñada bajo el enfoque **Local-First**. Es un entorno de grabación de estudio y teleprompter de nivel profesional optimizado para dispositivos móviles, el cual opera con cero dependencias de servidores externos, cero costos y total privacidad de datos.

### 🚀 Características Clave
* **Diseño Ergonómico de Foco Fijo:** Interfaz de lectura estática dividida en tres filas sólidas de alto contraste. Elimina el rastreo vertical de las pupilas, reduciendo la fatiga visual y garantizando una mirada natural a la cámara.
* **Scroll de Precisión Quirúrgica:** Calibración milimétrica de velocidad en pasos estrictos de **0.01** gestionados por `requestAnimationFrame` para un desplazamiento fluido a 60 FPS.
* **Estudio de Grabación Local (Web Audio API):** Puerta de ruido adaptativa integrada y filtro pasa-banda por hardware. Incluye una función de **costura digital por crossfade** automático que fusiona fragmentos de audio de forma invisible al pausar y reanudar tomas.
* **Sincronización Chroma Key (Modo Cine):** Sintoniza la reproducción del audio grabado con el desplazamiento del texto en horizontal sobre un fondo verde sólido (`#00FF00`). Diseñado para grabar la pantalla y superponer subtítulos dinámicos en editores como YouCut.
* **Privacidad Absoluta e Independencia:** Todos los datos, guiones y tomas de audio viven exclusivamente dentro del entorno seguro del navegador (IndexedDB / OPFS) con opción de exportación directa a la tarjeta SD externa.
